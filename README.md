📊 Blinkit Sales & Operations Analysis – Power BI Project
📌 Project Overview

This Power BI project analyzes the sales, product performance, and outlet operations data of Blinkit (India’s quick-commerce grocery delivery platform). The dashboard provides business insights into revenue trends, customer demand patterns, outlet efficiency, and category-level performance.

🎯 Business Objective

The objective of this project is to:

Analyze total sales and revenue trends

Identify top-performing product categories

Evaluate outlet performance by location and size

Understand customer purchasing behavior

Support data-driven decision-making for inventory and expansion strategy

📂 Dataset Description

The dataset includes:

Item Type

Item Fat Content

Outlet Location Type

Outlet Establishment Year

Sales Amount

Rating

Item Visibility

Outlet Size

📊 Key KPIs Used

💰 Total Sales

📦 Total Items Sold

⭐ Average Rating

📍 Sales by Outlet Location

🛒 Sales by Item Category

📈 Year-wise Outlet Performance

🛠 Tools & Techniques Used

Power BI

Data Cleaning in Power Query

Data Modeling

DAX Measures (SUM, AVERAGE, CALCULATE, FILTER)

Interactive Dashboard Design

📈 Key Insights

Tier 3 outlets generated higher total sales compared to Tier 1 in some cases.

Low-fat products contribute significantly to total revenue.

Certain product categories dominate overall sales performance.

Older outlets show stable revenue compared to newly established ones.

📷 Dashboard Features

Interactive slicers for outlet location and item type

KPI Cards for quick performance summary

Bar charts & pie charts for category comparison

Line chart for trend analysis

Drill-down capability

🚀 Conclusion

This dashboard helps understand operational efficiency and customer demand patterns for Blinkit’s retail model. It demonstrates strong data modeling, visualization, and business insight skills using Power BI.
